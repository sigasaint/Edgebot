Pro Tips

  • For best performance, only one control device should be connected at a time.
  • You can control EdgeBot from any browser — no app required.
  • For advanced users: read or edit main.py to tweak motor logic or add sensors.
  • Use the serial console to run commands like LIST, READ <filename>, and ID.
  • EdgeBot’s lightweight stack means low power, high speed, and easy debugging — perfect for education, R&D, or exploration.
  • The L298N module supports 2–4 motors depending on how you wire it — perfect for custom chassis setups!
