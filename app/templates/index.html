About EdgeBot

EdgeBot MK-II is a new-generation robotics system designed to be the official robotics standard for Zimbabwe. It is fully autonomous, runs on a microcontroller, and is completely controlled through lightweight Python and HTML systems.

Developed by Mthokozisi Thabiso Sapuwa under the Sentinel-X Systems R&D Lab, EdgeBot requires just two files: a main Python script and an HTML interface usable on phones and laptops.

Core Features

  • main.py — Full robot logic, AP mode server, motor control, and command interpreter
  • EdgeBot_MK-II_Control_Station.html — Clean UI for mobile and desktop use
  • Creates and hosts its own Wi-Fi network for control
  • Can also be controlled over external Wi-Fi if networked

System Benefits

  • Lightweight — only 2 files required to run
  • No Arduino IDE, no ArduPilot — 100% MicroPython
  • Designed for real-world use, learning, and modular upgrades
  • Field-tested and suitable for tactical missions, schools, and innovation hubs

Usage Flow

  1. Power on the EdgeBot
  2. Connect to its Wi-Fi hotspot (e.g., EdgeBot-MK2)
  3. Open http://192.168.4.1 in any browser
  4. Control the robot using the on-screen console
